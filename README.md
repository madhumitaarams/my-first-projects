# my-first-projects
Travel Management System

Overview
The Travel Management System is a Python-based application that allows users to manage travel destinations, make bookings, and view their upcoming trips. Admins can manage destinations, add or delete entries, and view all user bookings. Users can register, log in, and book trips to their desired destinations.

The system uses MySQL for database management and Python for implementing the backend logic. This project is designed for managing the travel workflow and can be extended to include more features like payment integration, reviews, etc.
Features
User Features:
- User Registration: Users can sign up with their name, email, and password.
- Login: Registered users can log in to access the system.
- View Destinations: Users can view available travel destinations.
- Make a Booking: Users can book trips to their selected destinations.

Login: Admins can log in with special credentials.
- Manage Destinations: Admins can add, delete, or update travel destinations.
- View All Bookings: Admins can view all bookings made by users.

Tech Stack
- Python: Main programming language.
- MySQL: For managing user, destination, and booking data.
- mysql-connector: Python library for interacting with MySQL.
